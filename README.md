# Nimauria Eggs

Production-grade Pterodactyl egg collection built for consistency, reliability, and maintainability.

Nimauria Eggs is a curated repository of Pterodactyl eggs designed to replace outdated, inconsistent, or poorly maintained community eggs with a clean, standardised system.

## 🧠 Project Goals

This repository is being built with a focus on:

* consistent structure across all eggs
* clean install and startup logic
* maintainable variable naming
* better documentation
* predictable behaviour
* long-term maintainability
